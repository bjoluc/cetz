<Function name="circle-through" parameters={{"a":{"types":"coordinate"},"b":{"types":"coordinate"},"c":{"types":"coordinate"},"name":{"default":"none","types":"none,str"},"anchor":{"default":"none","types":"none,str"},"..style":{"types":"style"}}}/>
Draws a circle through three coordinates.

```typc example
let (a, b, c) = ((0,0), (2,-.5), (1,1))
line(a, b, c, close: true, stroke: gray)
circle-through(a, b, c, name: "c")
circle("c.center", radius: .05, fill: red)
```

<Parameter name="a" types="coordinate" > Coordinate a.</Parameter>
<Parameter name="b" types="coordinate" > Coordinate b.</Parameter>
<Parameter name="c" types="coordinate" > Coordinate c.</Parameter>




### Styling
*Root*: `circle`

  `circle-through` has the same styling as [circle](#circle) except for `radius` as the circle's radius is calculated by the given coordinates.

### Anchors
Supports the same anchors as `circle` as well as:
- **a**: Coordinate a
- **b**: Coordinate b
- **c**: Coordinate c
