<Function name="circle" parameters={{"position":{"types":"coordinate"},"name":{"default":"none","types":"none,str"},"anchor":{"default":"none","types":"none, str"},"..style":{"types":"style"}}}/>
Draws a circle or ellipse.

```typc example
circle((0,0))
// Draws an ellipse
circle((0,-2), radius: (0.75, 0.5))
```

<Parameter name="position" types="coordinate" > The position to place the circle on.</Parameter>




### Styling
*Root*: `circle`

<Parameter name="radius" types="number, array" default_value="1"> A number that defines the size of the circle's radius. Can also be set to a tuple of two numbers to define the radii of an ellipse, the first number is the `x` radius and the second is the `y` radius.</Parameter>

### Anchors
  Supports border and path anchors. The "center" anchor is the default.
