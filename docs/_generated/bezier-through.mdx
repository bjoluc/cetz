<Function name="bezier-through" parameters={{"start":{"types":"coordinate"},"pass-through":{"types":"coordinate"},"end":{"types":"coordinate"},"name":{"default":"none","types":"none,str"},"..style":{"types":"style"}}}/>
Draws a cubic bezier curve through a set of three points. See `bezier` for style and anchor details.

```typc example
let (a, b, c) = ((0, 0), (1, 1), (2, -1))
line(a, b, c, stroke: gray)
bezier-through(a, b, c, name: "b")

// Show calculated control points
line(a, "b.ctrl-0", "b.ctrl-1", c, stroke: gray)
```

<Parameter name="start" types="coordinate" > The position to start the curve.</Parameter>
<Parameter name="pass-through" types="coordinate" > The position to pass the curve through.</Parameter>
<Parameter name="end" types="coordinate" > The position to end the curve.</Parameter>
