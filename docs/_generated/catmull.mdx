<Function name="catmull" parameters={{"..pts-style":{"types":"coordinate,style"},"close":{"default":"false","types":"bool"},"name":{"default":"none","types":"none,str"}}}/>
Draws a Catmull-Rom curve through a set of points.

```typc example
catmull((0,0), (1,1), (2,-1), (3,0), tension: .4, stroke: blue)
catmull((0,0), (1,1), (2,-1), (3,0), tension: .5, stroke: red)
```

<Parameter name="..pts-style" types="coordinate,style" > Positional arguments should be coordinates that the curve should pass through. Named arguments are for styling.</Parameter>
<Parameter name="close" types="bool" default_value="false"> Closes the curve with a straight line between the start and end of the curve.</Parameter>


## Styling 
*Root*: `catmull`

Supports marks.

<Parameter name="tension" types="float" default_value="0.5"> How tight the curve should fit to the points. The higher the tension the less curvy the curve.</Parameter>

## Anchors
Supports path anchors.
- **pt-n**: The nth given position (0 indexed so "pt-0" is equal to "start")
